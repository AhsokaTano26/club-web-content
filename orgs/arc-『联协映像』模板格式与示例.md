# 『联协映像』文档编写模板与示例

欢迎使用联协映像官方文档模板。本模板展示了如何在 Markdown 文档中插入图片、创建表格、模拟推文风格等常见需求。请根据实际内容修改 Frontmatter 及正文。

---

## 📌 Frontmatter 字段说明

每个文档开头必须包含 YAML 格式的元数据块，字段如下：

| 字段 | 类型 | 说明 | 示例 |
| :--- | :--- | :--- | :--- |
| `title` | string | 文档标题 | `"『画师图集』跨世界同好会2026年记录"` |
| `description` | string | 简短描述，用于摘要或 SEO | `"秦大小姐本月出图"` |
| `orgs` | string | 所属组织，多个用斜杠 `/` 分隔 | `"跨世界同好会 / WTW団"` |
| `date` | YYYY-MM-DD | 发布日期 | `2026-03-28` |
| `author` | string | 作者姓名或昵称 | `"Star"` |
| `type` | string | 文档类型：`Gallery`（图集）、`tweet`（推文存档）、`article`（长文） | `"Gallery"` |

📂 图片路径约定
建议将图片统一放在文档同级目录下的 assets/ 文件夹中。

示例路径：assets/2043253.jpg、assets/orgs/test.jpg。

若图片仅供占位，可保留原示例文件名，最终替换为实际图片。
